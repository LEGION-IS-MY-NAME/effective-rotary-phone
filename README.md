# Android Console Home Assistant Add-ons

This repository packages the **Android Console (USB scrcpy)** add-on so you can mirror and control Android devices connected via USB directly inside Home Assistant through Ingress.

## Add the repository
1. Open **Settings → Add-ons → Add-on store** in Home Assistant.
2. Click the kebab menu (⋮) → **Repositories**.
3. Add this repository URL.
4. Install **Android Console (USB scrcpy)** and start it.

See [`android_console/DOCS.md`](android_console/DOCS.md) for usage details, configuration, and troubleshooting.
